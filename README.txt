Подключение класса:
require_once( "simplelog.class.php" );

Настройки:
$simplelog_config = array(
"type" => "stdout", //db OR file OR stdout (в зависимости от типа вывода)

"db_host" => "localhost", //сервер подключения
"db_name" => "simplelog", //имя базы
"db_user" => "simplelog", //имя пользователя
"db_pass" => "123456", //пароль пользователя

"file" => dirname( __FILE__ ) . "/simple_log.txt", //файл для записи логов

"compile" => true, //если true, то все логи добавляются одним запросом, в конце выполнения
);

Создание:
$simple_log = new SimpleLog( $simplelog_config );

Добавление лога:
$simple_log->add( $log );

$log может являться массивом, строкой, объектом или же исключением.
Каждый лог по умолчанию добавляется сразу после выполнения функции add (на каждый лог идет отдельный запрос к бд или к файлу).
Если в настройках значение ключа compile равняется true, то все логи добавляются в конце выполнения.


Запрос для создания базы данных находится в файле db.sql (для записи логов в БД).
